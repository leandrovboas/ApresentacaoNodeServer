Node.Js Como Plataforma Corportiva

O que é o Node.js

Node.js não é uma linguagem de programação

Node.js não é um framework Javascript

Node.js está mais para uma plataforma de aplicação, na qual você escreve seus programas com Javascript que serão compilados, otimizados e interpretados pela máquina virtual V8. Essa VM é a mesma que o Google utiliza para executar Javascript no browser Chrome.

Porque o Node.js foi Criado

Node.js foi originalmente escrito por Ryan Dahl, em 2009.

Ele conta em diversas entrevistas que foi inspirado a criar o Node.js após ver uma barra de progresso de upload no Flickr.

Ele entendeu que o navegador tinha que ficar perguntando para o servidor quanto do arquivo faltava a ser transmitido, pois ele não tinha essa informação, e que isso era um desperdício de tempo e recursos.

Ele queria criar um jeito mais fácil de fazer isso.

Porque usar Node.js

  • JavaScript - Linguagem de programação mais popular no mundo
  • Node.js permite Javascript full-stack - Ele permitir que você trabalhe com JS no Front-end, Back-end, banco de dados e mobile
  • Open-source - Tendo como suporte para plataforma a Node.org e a Node Foundation
  • multiplataforma - Isso permite que você consiga rodar seus projetos em servidores abertos e com o SO que quiser

Ambiente de Desenvolvimento

O Node.js é composto basicamente por:

  • > Um runtime JavaScript (Google V8, o mesmo do Chrome);
  • > Uma biblioteca para I/O de baixo nível (libuv);
  • > Bibliotecas de desenvolvimento básico (os core modules);
  • > Um gerenciador de pacotes via linha de comando (NPM);
  • > Um gerenciador de versões via linha de comando (NVM);
  • > Utilitário REPL via linha de comando;
